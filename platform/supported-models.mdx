## Overview

OpenModex provides access to 100+ models from 6 major AI providers through a single API. All models are accessible via the same OpenAI-compatible endpoint.

<Tip>
  Use the [Models API](/api-reference/models-list) to get real-time pricing and availability, or the [Compare API](/api-reference/models-compare) to evaluate models side by side.
</Tip>

## Choosing a Model

Use the routing strategies or refer to this quick guide:

| Priority | Recommended |
|----------|-------------|
| **Cheapest** | `gpt-4o-mini`, `deepseek-chat`, `gemini-2.0-flash` |
| **Fastest** | `gemini-2.0-flash`, `gpt-4o-mini`, `claude-3-haiku` |
| **Best quality** | `gpt-4o`, `claude-3.5-sonnet`, `gemini-1.5-pro` |
| **Best at code** | `claude-3.5-sonnet`, `deepseek-reasoner`, `codestral` |
| **Longest context** | `gemini-1.5-pro` (2M), `gemini-2.0-flash` (1M) |
| **Best reasoning** | `o3`, `o1`, `deepseek-reasoner` |

Or let OpenModex decide automatically:

```typescript
const response = await client.chat.completions.create({
  model: 'auto',
  messages: [{ role: 'user', content: 'Hello!' }],
  routing: { strategy: 'cost_optimized' },
});
```

## Live Model Data

For real-time pricing, availability, and quality scores:

```bash
# List all models with pricing
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"

# Compare specific models
curl "https://api.openmodex.com/v1/models/compare?models=openai/gpt-4o,anthropic/claude-3.5-sonnet" \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

## How Pricing Works

OpenModex uses **pay-as-you-go** pricing. You're billed only for the tokens you consume, at transparent pass-through rates.

<Info>
  OpenModex charges the same per-token rate as the underlying provider. There are no hidden markups on model costs.
</Info>

## Cost Tracking

Every API response includes cost data:

```json
{
  "usage": {
    "prompt_tokens": 150,
    "completion_tokens": 50,
    "total_tokens": 200,

  }
}
```

<Note>
  Le CLI OpenModex est actuellement en phase de développement initial. Les fonctionnalités principales arrivent bientôt.
</Note>

## Installation

```bash
npm install -g @openmodex/cli
```

## Utilisation

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## Fonctionnalités prévues

| Commande | Description | Statut |
|---------|-------------|--------|
| `openmodex chat` | Chat interactif depuis le terminal | Bientôt disponible |
| `openmodex models` | Lister et comparer les modèles | Bientôt disponible |
| `openmodex config` | Gérer les clés API et les paramètres | Bientôt disponible |
| `openmodex playground` | Test rapide de prompts | Bientôt disponible |

## Installation

```bash
npm install @openmodex/sdk
```

Nécessite **Node.js 18+**.

## Démarrage rapide

```typescript
import OpenModex from '@openmodex/sdk';

const client = new OpenModex({
  apiKey: process.env.OPENMODEX_API_KEY,
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Hello!' }],
});

console.log(response.choices[0].message.content);
```

## Configuration

```typescript
const client = new OpenModex({
  apiKey: 'omx_sk_...',
  baseURL: 'https://api.openmodex.com/v1',  // default
  timeout: 60_000,                           // default: 30000
  maxRetries: 3,                             // default: 2
  defaultModel: 'gpt-4o',
  fallbackModels: ['claude-3.5-sonnet'],
});
```

| Option | Type | Défaut | Description |
|--------|------|---------|-------------|
| `apiKey` | `string` | env `OPENMODEX_API_KEY` | Clé API |
| `baseURL` | `string` | `https://api.openmodex.com/v1` | URL de base de l'API |
| `timeout` | `number` | `30000` | Timeout de requête (ms) |
| `maxRetries` | `number` | `2` | Nombre de nouvelles tentatives automatiques |
| `defaultModel` | `string` | — | Modèle par défaut |
| `fallbackModels` | `string[]` | `[]` | Chaîne de modèles de basculement |
| `defaultHeaders` | `object` | `{}` | En-têtes personnalisés |
| `fetch` | `typeof fetch` | `globalThis.fetch` | Fetch personnalisé |

## Chat Completions

```typescript
const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [
    { role: 'system', content: 'You are a helpful assistant.' },
    { role: 'user', content: 'What is TypeScript?' },
  ],
  temperature: 0.7,
  max_tokens: 500,
});
```

## Streaming

```typescript
const stream = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [{ role: 'user', content: 'Write a poem.' }],
  stream: true,
});

for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0]?.delta?.content ?? '');
}

// Abort a stream
stream.abort();
```

## Modèles

```typescript
// List all models
const models = await client.models.list();

// Get model details
const model = await client.models.retrieve('openai/gpt-4o');

// Compare models
const comparison = await client.models.compare(['openai/gpt-4o', 'anthropic/claude-3.5-sonnet']);
```

## Gestion des erreurs

```typescript
import { APIError } from '@openmodex/sdk';

try {
  await client.chat.completions.create({...});
} catch (err) {
  if (err instanceof APIError) {
    console.log(err.statusCode, err.message);
    console.log(err.isRateLimited);   // true if 429
    console.log(err.isAuthError);     // true if 401
    console.log(err.isServerError);   // true if 5xx
  }
}
```

## Compatibilité SDK OpenAI

Vous utilisez déjà le SDK OpenAI ? Vous pouvez l'utiliser avec OpenModex en changeant l'URL de base :

<CodeGroup>
```typescript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',
  baseURL: 'https://api.openmodex.com/v1',
});
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="omx_sk_...",
    base_url="https://api.openmodex.com/v1",
)
```
</CodeGroup>

## Vue d'ensemble

OpenModex donne accès à plus de 100 modèles de 6 fournisseurs d'IA majeurs via une seule API. Tous les modèles sont accessibles via le même endpoint compatible OpenAI.

<Tip>
  Utilisez l'[API Models](/fr/api-reference/models-list) pour obtenir les tarifs et la disponibilité en temps réel, ou l'[API Compare](/fr/api-reference/models-compare) pour évaluer les modèles côte à côte.
</Tip>

## Choisir un modèle

Utilisez les stratégies de routage ou consultez ce guide rapide :

| Priorité | Recommandé |
|----------|-------------|
| **Le moins cher** | `gpt-4o-mini`, `deepseek-chat`, `gemini-2.0-flash` |
| **Le plus rapide** | `gemini-2.0-flash`, `gpt-4o-mini`, `claude-3-haiku` |
| **Meilleure qualité** | `gpt-4o`, `claude-3.5-sonnet`, `gemini-1.5-pro` |
| **Meilleur en code** | `claude-3.5-sonnet`, `deepseek-reasoner`, `codestral` |
| **Plus long contexte** | `gemini-1.5-pro` (2M), `gemini-2.0-flash` (1M) |
| **Meilleur raisonnement** | `o3`, `o1`, `deepseek-reasoner` |

Ou laissez OpenModex décider automatiquement :

```typescript
const response = await client.chat.completions.create({
  model: 'auto',
  messages: [{ role: 'user', content: 'Hello!' }],
  routing: { strategy: 'cost_optimized' },
});
```

## Données en temps réel sur les modèles

Pour les tarifs, la disponibilité et les scores de qualité en temps réel :

```bash
# List all models with pricing
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"

# Compare specific models
curl "https://api.openmodex.com/v1/models/compare?models=openai/gpt-4o,anthropic/claude-3.5-sonnet" \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

## 概要

OpenModexは、単一のAPIを通じて6つの主要AIプロバイダーから100以上のモデルへのアクセスを提供します。すべてのモデルは同じOpenAI互換エンドポイントからアクセスできます。

<Tip>
  リアルタイムの料金と可用性については[モデルAPI](/ja/api-reference/models-list)を、モデルの並列評価には[比較API](/ja/api-reference/models-compare)をご利用ください。
</Tip>

## モデルの選び方

ルーティング戦略を使用するか、以下のクイックガイドを参考にしてください：

| 優先事項 | 推奨 |
|----------|-------------|
| **最安値** | `gpt-4o-mini`, `deepseek-chat`, `gemini-2.0-flash` |
| **最速** | `gemini-2.0-flash`, `gpt-4o-mini`, `claude-3-haiku` |
| **最高品質** | `gpt-4o`, `claude-3.5-sonnet`, `gemini-1.5-pro` |
| **コードに最適** | `claude-3.5-sonnet`, `deepseek-reasoner`, `codestral` |
| **最長コンテキスト** | `gemini-1.5-pro` (2M), `gemini-2.0-flash` (1M) |
| **最高の推論** | `o3`, `o1`, `deepseek-reasoner` |

または、OpenModexに自動的に判断させることもできます：

```typescript
const response = await client.chat.completions.create({
  model: 'auto',
  messages: [{ role: 'user', content: 'Hello!' }],
  routing: { strategy: 'cost_optimized' },
});
```

## ライブモデルデータ

リアルタイムの料金、可用性、品質スコアについては：

```bash
# List all models with pricing
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"

# Compare specific models
curl "https://api.openmodex.com/v1/models/compare?models=openai/gpt-4o,anthropic/claude-3.5-sonnet" \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

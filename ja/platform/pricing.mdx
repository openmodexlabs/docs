## 料金の仕組み

OpenModexは**従量課金制**の料金体系です。消費したトークンに対してのみ、透明なパススルーレートで課金されます。

<Info>
  OpenModexは基盤となるプロバイダーと同じトークン単価を請求します。モデルコストに隠れたマークアップはありません。
</Info>

| **フォールバック** | 1レベル | 無制限 | 無制限 |

| **サポート** | コミュニティ | メール | 専任 |
| **アナリティクス** | 基本 | フル | フル + エクスポート |
| **チーム管理** | — | 最大5名 | 無制限 |

## トークン料金

モデルコストはプロバイダーレートでパススルーされます。リアルタイムの料金は[モデルAPI](/ja/api-reference/models-list)または[ダッシュボード](https://openmodex.com/dashboard)でご確認ください。

### キャッシュされたレスポンス

プロンプトキャッシュによりコストが大幅に削減されます：

| | 標準 | キャッシュ |
|--|--------:|-------:|
| 入力トークン | 100% | 約50% |
| 出力トークン | 100% | $0 |
| レイテンシー | 通常 | &lt;10ms |

## コスト追跡

すべてのAPIレスポンスにコストデータが含まれます：

```json
{
  "usage": {
    "prompt_tokens": 150,
    "completion_tokens": 50,
    "total_tokens": 200,
    "cost_usd": 0.001
  }
}
```

## 残高と支払い

- **クレジットカード**（Stripe）または**暗号通貨**で残高を追加
- [ダッシュボード](https://openmodex.com/dashboard)で残高と取引履歴を確認
- 残高不足の通知用に**アラートを設定**
- ProおよびEnterpriseティアで**自動チャージ**が利用可能

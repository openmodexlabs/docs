  </Card>
  <Card title="Python" icon="python" href="/ja/sdks/python">

  <Card title="Go" icon="golang" href="/ja/sdks/go">

## OpenAI SDK互換性

すでにOpenAI SDKを使用していますか？ベースURLを変更するだけでOpenModexと使用できます：

<CodeGroup>
```typescript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',
  baseURL: 'https://api.openmodex.com/v1',
});
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="omx_sk_...",
    base_url="https://api.openmodex.com/v1",
)
```
</CodeGroup>

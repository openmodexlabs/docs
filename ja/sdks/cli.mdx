<Note>
  OpenModex CLIは現在初期開発中です。コア機能は近日公開予定です。
</Note>

## インストール

```bash
npm install -g @openmodex/cli
```

## 使用方法

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## 予定されている機能

| コマンド | 説明 | ステータス |
|---------|-------------|--------|
| `openmodex chat` | ターミナルからのインタラクティブチャット | 近日公開 |
| `openmodex models` | モデルの一覧表示と比較 | 近日公開 |
| `openmodex config` | APIキーと設定の管理 | 近日公開 |
| `openmodex playground` | クイックプロンプトテスト | 近日公開 |

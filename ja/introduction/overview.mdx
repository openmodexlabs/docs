## ユニバーサルAIゲートウェイ

OpenModexは、すべての主要プロバイダーの100以上の大規模言語モデルに、単一のOpenAI互換APIを通じてアクセスできる統合APIゲートウェイです。

複数のAPIキー、異なるSDK、各AIプロバイダーごとの別々の課金を管理する代わりに、OpenModexでは以下のことが可能です：

- **1つのAPIキー**でOpenAI、Anthropic、Google Gemini、DeepSeek、Mistral、Qwenなどにアクセス
- **モデルを即座に切り替え** — コードベース全体ではなく、1つのパラメータを変更するだけ
- **インテリジェントなルーティング** — コスト、レイテンシー、品質に基づいてOpenModexが最適なモデルを選択
- **グレースフルなフェイルオーバー** — 自動フォールバックにより、プロバイダーがダウンしてもアプリが稼働し続けます
- **コスト削減** — プロンプトキャッシュ、コスト最適化ルーティング、透明性のある料金体系

## 仕組み

<Steps>
  <Step title="APIキーを取得">
    [openmodex.com](https://openmodex.com)でサインアップし、ダッシュボードからAPIキーを作成します。
  </Step>
  <Step title="コードをOpenModexに向ける">
    プロバイダーのベースURLを`https://api.openmodex.com/v1`に置き換えます。すでにOpenAI SDKを使用している場合は、1行の変更で済みます。
  </Step>
  <Step title="任意のモデルを選択">
    `gpt-4o`、`claude-3.5-sonnet`、`gemini-2.0-flash`、`deepseek-chat`など、サポートされている任意のモデルを名前で使用できます。
  </Step>
</Steps>

## OpenAI互換

すでにOpenAI APIを使用している場合、OpenModexへの切り替えは簡単です：

```typescript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',                      // your OpenModex key
  baseURL: 'https://api.openmodex.com/v1',   // point to OpenModex
});

// Everything else stays the same
const response = await client.chat.completions.create({
  model: 'gpt-4o',  // or any other model
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

<Note>
  OpenModex CLI 目前處於早期開發階段。核心功能即將推出。
</Note>

## 安裝

```bash
npm install -g @openmodex/cli
```

## 使用方式

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## 計劃中的功能

| 指令 | 說明 | 狀態 |
|---------|-------------|--------|
| `openmodex chat` | 從終端機進行互動式聊天 | 即將推出 |
| `openmodex models` | 列出和比較模型 | 即將推出 |
| `openmodex config` | 管理 API 金鑰和設定 | 即將推出 |
| `openmodex playground` | 快速提示測試 | 即將推出 |

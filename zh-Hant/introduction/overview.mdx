## 通用 AI 閘道器

OpenModex 是一個統一的 API 閘道器，讓您透過單一的 OpenAI 相容 API 存取來自所有主要供應商的 100+ 大型語言模型。

不再需要管理多個 API 金鑰、不同的 SDK 和各個 AI 供應商的獨立帳單，OpenModex 讓您可以：

- **使用一個 API 金鑰**存取 OpenAI、Anthropic、Google Gemini、DeepSeek、Mistral、Qwen 等
- **即時切換模型** — 只需更改一個參數，無需改動整個程式碼庫
- **智慧路由** — 讓 OpenModex 根據成本、延遲或品質自動選擇最佳模型
- **優雅降級** — 自動容錯備援確保即使供應商中斷，您的應用程式也能正常運作
- **節省成本** — 提示快取、成本最佳化路由和透明定價

## 運作方式

<Steps>
  <Step title="取得您的 API 金鑰">
    在 [openmodex.com](https://openmodex.com) 註冊，並在控制台建立 API 金鑰。
  </Step>
  <Step title="將程式碼指向 OpenModex">
    將供應商的基底 URL 替換為 `https://api.openmodex.com/v1`。如果您已在使用 OpenAI SDK，只需更改一行程式碼。
  </Step>
  <Step title="選擇任何模型">
    直接使用模型名稱 — `gpt-4o`、`claude-3.5-sonnet`、`gemini-2.0-flash`、`deepseek-chat` 等眾多模型。
  </Step>
</Steps>

## OpenAI 相容

如果您已在使用 OpenAI API，切換到 OpenModex 非常簡單：

```typescript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',                      // your OpenModex key
  baseURL: 'https://api.openmodex.com/v1',   // point to OpenModex
});

// Everything else stays the same
const response = await client.chat.completions.create({
  model: 'gpt-4o',  // or any other model
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

## 概覽

OpenModex 透過單一 API 提供來自 6 家主要 AI 供應商的 100+ 模型存取。所有模型都可透過相同的 OpenAI 相容端點存取。

<Tip>
  使用 [Models API](/zh-Hant/api-reference/models-list) 取得即時定價和可用性資訊，或使用 [Compare API](/zh-Hant/api-reference/models-compare) 並排評估模型。
</Tip>

## 選擇模型

使用路由策略或參考此快速指南：

| 優先順序 | 推薦 |
|----------|-------------|
| **最便宜** | `gpt-4o-mini`, `deepseek-chat`, `gemini-2.0-flash` |
| **最快** | `gemini-2.0-flash`, `gpt-4o-mini`, `claude-3-haiku` |
| **最佳品質** | `gpt-4o`, `claude-3.5-sonnet`, `gemini-1.5-pro` |
| **最擅長程式碼** | `claude-3.5-sonnet`, `deepseek-reasoner`, `codestral` |
| **最長上下文** | `gemini-1.5-pro` (2M), `gemini-2.0-flash` (1M) |
| **最佳推理** | `o3`, `o1`, `deepseek-reasoner` |

或讓 OpenModex 自動決定：

```typescript
const response = await client.chat.completions.create({
  model: 'auto',
  messages: [{ role: 'user', content: 'Hello!' }],
  routing: { strategy: 'cost_optimized' },
});
```

## 即時模型資料

取得即時定價、可用性和品質評分：

```bash
# List all models with pricing
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"

# Compare specific models
curl "https://api.openmodex.com/v1/models/compare?models=openai/gpt-4o,anthropic/claude-3.5-sonnet" \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

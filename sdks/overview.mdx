## OpenAI SDK Compatibility

Already using the OpenAI SDK? You can use it with OpenModex by changing the base URL:

<CodeGroup>
```typescript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',
  baseURL: 'https://api.openmodex.com/v1',
});
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="omx_sk_...",
    base_url="https://api.openmodex.com/v1",
)
```

```rust Rust
use openmodex::OpenModex;

let client = OpenModex::builder()
    .api_key("omx_sk_...")
    .base_url("https://api.openmodex.com/v1")
    .build()?;
```
</CodeGroup>

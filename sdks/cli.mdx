---
title: "CLI"
description: "Command-line interface for the OpenModex API."
---

<Note>
  The OpenModex CLI is currently in early development. Core features are coming soon.
</Note>

## Installation

```bash
npm install -g @openmodex/cli
```

## Usage

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## Planned Features

| Command | Description | Status |
|---------|-------------|--------|
| `openmodex chat` | Interactive chat from terminal | Coming Soon |
| `openmodex models` | List and compare models | Coming Soon |
| `openmodex config` | Manage API keys and settings | Coming Soon |
| `openmodex playground` | Quick prompt testing | Coming Soon |

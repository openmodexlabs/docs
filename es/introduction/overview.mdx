## La Pasarela Universal de IA

OpenModex es una pasarela API unificada que le brinda acceso a más de 100 modelos de lenguaje grande de todos los proveedores principales a través de una única API compatible con OpenAI.

En lugar de gestionar múltiples claves API, diferentes SDKs y facturación separada para cada proveedor de IA, OpenModex le permite:

- **Usar una sola clave API** para acceder a OpenAI, Anthropic, Google Gemini, DeepSeek, Mistral, Qwen y más
- **Cambiar de modelo al instante** — modifique un parámetro, no todo su código
- **Enrutar de forma inteligente** — deje que OpenModex elija el mejor modelo según costo, latencia o calidad
- **Fallar con elegancia** — los respaldos automáticos aseguran que su aplicación siga funcionando incluso cuando los proveedores caen
- **Ahorrar dinero** — almacenamiento en caché de prompts, enrutamiento optimizado por costo y precios transparentes

## Cómo Funciona

<Steps>
  <Step title="Obtenga su clave API">
    Regístrese en [openmodex.com](https://openmodex.com) y cree una clave API desde su panel de control.
  </Step>
  <Step title="Apunte su código a OpenModex">
    Reemplace la URL base de su proveedor con `https://api.openmodex.com/v1`. Si ya está usando el SDK de OpenAI, es un cambio de una sola línea.
  </Step>
  <Step title="Elija cualquier modelo">
    Use cualquier modelo compatible por nombre — `gpt-4o`, `claude-3.5-sonnet`, `gemini-2.0-flash`, `deepseek-chat` y muchos más.
  </Step>
</Steps>

## Compatible con OpenAI

Si ya está usando la API de OpenAI, cambiar a OpenModex es trivial:

```typescript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',                      // your OpenModex key
  baseURL: 'https://api.openmodex.com/v1',   // point to OpenModex
});

// Everything else stays the same
const response = await client.chat.completions.create({
  model: 'gpt-4o',  // or any other model
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

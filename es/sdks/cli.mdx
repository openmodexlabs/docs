<Note>
  El CLI de OpenModex se encuentra actualmente en desarrollo temprano. Las funciones principales estarán disponibles próximamente.
</Note>

## Instalación

```bash
npm install -g @openmodex/cli
```

## Uso

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## Descripción General

OpenModex proporciona acceso a más de 100 modelos de 6 proveedores principales de IA a través de una única API. Todos los modelos son accesibles mediante el mismo endpoint compatible con OpenAI.

<Tip>
  Use la [API de Modelos](/es/api-reference/models-list) para obtener precios y disponibilidad en tiempo real, o la [API de Comparación](/es/api-reference/models-compare) para evaluar modelos lado a lado.
</Tip>

## Elegir un Modelo

Use las estrategias de enrutamiento o consulte esta guía rápida:

| Prioridad | Recomendado |
|----------|-------------|
| **Más económico** | `gpt-4o-mini`, `deepseek-chat`, `gemini-2.0-flash` |
| **Más rápido** | `gemini-2.0-flash`, `gpt-4o-mini`, `claude-3-haiku` |
| **Mejor calidad** | `gpt-4o`, `claude-3.5-sonnet`, `gemini-1.5-pro` |
| **Mejor en código** | `claude-3.5-sonnet`, `deepseek-reasoner`, `codestral` |
| **Contexto más largo** | `gemini-1.5-pro` (2M), `gemini-2.0-flash` (1M) |
| **Mejor razonamiento** | `o3`, `o1`, `deepseek-reasoner` |

O deje que OpenModex decida automáticamente:

```typescript
const response = await client.chat.completions.create({
  model: 'auto',
  messages: [{ role: 'user', content: 'Hello!' }],
  routing: { strategy: 'cost_optimized' },
});
```

## Datos de Modelos en Vivo

Para precios en tiempo real, disponibilidad y puntuaciones de calidad:

```bash
# List all models with pricing
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"

# Compare specific models
curl "https://api.openmodex.com/v1/models/compare?models=openai/gpt-4o,anthropic/claude-3.5-sonnet" \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

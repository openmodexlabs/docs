## Cách Tính Giá

OpenModex sử dụng mô hình **thanh toán theo mức sử dụng**. Bạn chỉ bị tính phí cho số token bạn tiêu thụ, với mức giá chuyển tiếp minh bạch.

<Info>
  OpenModex tính cùng mức giá theo token như nhà cung cấp gốc. Không có phí ẩn nào đối với chi phí mô hình.
</Info>

## Giá Token

Chi phí mô hình được chuyển tiếp theo giá nhà cung cấp. Kiểm tra giá thời gian thực qua [API Mô hình](/vi/api-reference/models-list) hoặc [Bảng điều khiển](https://openmodex.com/dashboard).

### Phản Hồi Từ Bộ Đệm

Bộ nhớ đệm prompt giảm chi phí đáng kể:

| | Tiêu chuẩn | Từ bộ đệm |
|--|--------:|-------:|
| Token đầu vào | 100% | ~50% |
| Token đầu ra | 100% | $0 |
| Độ trễ | Bình thường | &lt;10ms |

## Theo Dõi Chi Phí

Mỗi phản hồi API đều bao gồm dữ liệu chi phí:

```json
{
  "usage": {
    "prompt_tokens": 150,
    "completion_tokens": 50,
    "total_tokens": 200,
    "cost_usd": 0.001
  }
}
```

## Số Dư và Thanh Toán

- **Nạp tiền** qua thẻ tín dụng (Stripe) hoặc tiền mã hóa
- **Xem số dư** và lịch sử giao dịch trong [Bảng điều khiển](https://openmodex.com/dashboard)
- **Đặt cảnh báo** cho thông báo số dư thấp
- **Tự động nạp tiền** có sẵn cho gói Pro và Enterprise

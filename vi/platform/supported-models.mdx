## Tổng Quan

OpenModex cung cấp quyền truy cập vào hơn 100 mô hình từ 6 nhà cung cấp AI hàng đầu thông qua một API duy nhất. Tất cả mô hình đều có thể truy cập qua cùng một endpoint tương thích với OpenAI.

<Tip>
  Sử dụng [API Mô hình](/vi/api-reference/models-list) để xem giá và tình trạng khả dụng theo thời gian thực, hoặc [API So sánh](/vi/api-reference/models-compare) để đánh giá các mô hình song song.
</Tip>

## Chọn Mô Hình

Sử dụng các chiến lược định tuyến hoặc tham khảo hướng dẫn nhanh này:

| Ưu tiên | Đề xuất |
|----------|-------------|
| **Rẻ nhất** | `gpt-4o-mini`, `deepseek-chat`, `gemini-2.0-flash` |
| **Nhanh nhất** | `gemini-2.0-flash`, `gpt-4o-mini`, `claude-3-haiku` |
| **Chất lượng tốt nhất** | `gpt-4o`, `claude-3.5-sonnet`, `gemini-1.5-pro` |
| **Lập trình tốt nhất** | `claude-3.5-sonnet`, `deepseek-reasoner`, `codestral` |
| **Ngữ cảnh dài nhất** | `gemini-1.5-pro` (2M), `gemini-2.0-flash` (1M) |
| **Suy luận tốt nhất** | `o3`, `o1`, `deepseek-reasoner` |

Hoặc để OpenModex tự động quyết định:

```typescript
const response = await client.chat.completions.create({
  model: 'auto',
  messages: [{ role: 'user', content: 'Hello!' }],
  routing: { strategy: 'cost_optimized' },
});
```

## Dữ Liệu Mô Hình Thời Gian Thực

Để xem giá, tình trạng khả dụng và điểm chất lượng theo thời gian thực:

```bash
# List all models with pricing
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"

# Compare specific models
curl "https://api.openmodex.com/v1/models/compare?models=openai/gpt-4o,anthropic/claude-3.5-sonnet" \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

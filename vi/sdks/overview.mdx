<CardGroup cols={2}>

  </Card>
  <Card title="Python" icon="python" href="/vi/sdks/python">
    `pip install openmodex`
  </Card>
  <Card title="Go" icon="golang" href="/vi/sdks/go">

  </Card>
  <Card title="React" icon="react" href="/vi/sdks/react">
    `npm install @openmodex/react`
  </Card>

## Tương Thích OpenAI SDK

Đã sử dụng OpenAI SDK? Bạn có thể sử dụng nó với OpenModex bằng cách thay đổi URL gốc:

<CodeGroup>
```typescript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',
  baseURL: 'https://api.openmodex.com/v1',
});
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="omx_sk_...",
    base_url="https://api.openmodex.com/v1",
)
```
</CodeGroup>

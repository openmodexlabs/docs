<Note>
  OpenModex CLI hiện đang trong giai đoạn phát triển sớm. Các tính năng cốt lõi sẽ sớm ra mắt.
</Note>

## Cài Đặt

```bash
npm install -g @openmodex/cli
```

## Cách Sử Dụng

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## Tính Năng Dự Kiến

| Lệnh | Mô tả | Trạng thái |
|---------|-------------|--------|
| `openmodex chat` | Chat tương tác từ terminal | Sắp ra mắt |
| `openmodex models` | Liệt kê và so sánh mô hình | Sắp ra mắt |
| `openmodex config` | Quản lý khóa API và cài đặt | Sắp ra mắt |
| `openmodex playground` | Kiểm tra prompt nhanh | Sắp ra mắt |

<Note>
  Đây là endpoint cũ. Đối với hầu hết các trường hợp sử dụng, hãy sử dụng [Hoàn thành Chat](/vi/api-reference/chat-completions) thay thế.
</Note>

## Thân Yêu Cầu

<ParamField body="model" type="string" required>
  ID mô hình để sử dụng.
</ParamField>

<ParamField body="prompt" type="string | string[]" required>
  (Các) prompt để tạo hoàn thành.
</ParamField>

<ParamField body="max_tokens" type="integer" default={16}>
  Số lượng token tối đa để tạo.
</ParamField>

<ParamField body="temperature" type="number" default={1}>
  Nhiệt độ lấy mẫu từ 0 đến 2.
</ParamField>

<ParamField body="top_p" type="number" default={1}>
  Tham số lấy mẫu nucleus.
</ParamField>

<ParamField body="stop" type="string | string[]">
  (Các) chuỗi dừng.
</ParamField>

<ParamField body="suffix" type="string">
  Văn bản để thêm vào sau phần hoàn thành.
</ParamField>

## Phản Hồi

```json
{
  "id": "cmpl-abc123",
  "object": "text_completion",
  "created": 1700000000,
  "model": "gpt-3.5-turbo-instruct",
  "choices": [
    {
      "index": 0,
      "text": " a time, in a land far away...",
      "finish_reason": "length"
    }
  ],
  "usage": {
    "prompt_tokens": 5,
    "completion_tokens": 10,
    "total_tokens": 15
  }
}
```

## Ví Dụ

```bash
curl https://api.openmodex.com/v1/completions \
  -H "Authorization: Bearer omx_sk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-3.5-turbo-instruct",
    "prompt": "Once upon",
    "max_tokens": 100
  }'
```

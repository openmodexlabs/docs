## 2026년 3월

### v1.4.0 — 모델 비교 API

<Info>2026년 3월 15일 출시</Info>

- **신규:** 모델 간 비교 평가를 위한 `GET /v1/models/compare` 엔드포인트
- **신규:** 모델 응답의 품질 점수 (종합, 코드, 수학, 추론)
- **신규:** 모델별 성능 지표 (지연 시간 p50/p99, 가용성)
- **개선:** 모델 목록에 가격 내역 포함 (공식 가격 대 OpenModex 요금)

### v1.3.0 — BYOK 지원

<Info>2026년 3월 1일 출시</Info>

- **신규:** 자체 키 사용 — 자체 제공업체 API 키 사용 가능
- **신규:** 제공업체 키 저장을 위한 AWS KMS 봉투 암호화
- **신규:** 대시보드에서 제공업체 키 관리
- **개선:** 라우팅 엔진이 혼합 BYOK + 관리 설정을 처리

## 2026년 2월

### v1.2.0 — React 훅

<Info>2026년 2월 15일 출시</Info>

- **신규:** `useChat`, `useCompletion`, `useModels` 훅이 포함된 `@openmodex/react` 패키지
- **신규:** React 훅에 내장 스트리밍 및 중단 지원
- **개선:** TypeScript SDK에서 모든 타입 내보내기

### v1.1.0 — 프롬프트 캐싱

<Info>2026년 2월 1일 출시</Info>

- **신규:** 설정 가능한 TTL (60초~24시간)의 서버 측 프롬프트 캐싱
- **신규:** 응답 메타데이터의 `cache_hit` 필드
- **개선:** 캐시 응답은 입력 비용의 50%로 청구, 출력 비용 없음

## 가격 정책 작동 방식

OpenModex는 **종량제** 가격 정책을 사용합니다. 소비한 토큰에 대해서만 투명한 패스스루 요금으로 청구됩니다.

<Info>
  OpenModex는 기본 제공업체와 동일한 토큰당 요금을 부과합니다. 모델 비용에 대한 숨겨진 마크업이 없습니다.
</Info>

## 구독 티어

| | Free | Pro | Enterprise |
|--|:---:|:---:|:---:|
| **가격** | $0/월 | $29/월 | 커스텀 |
| **분당 요청** | 30 | 120 | 300+ |
| **분당 토큰** | 60,000 | 500,000 | 커스텀 |
| **모델** | 전체 | 전체 | 전체 + 커스텀 |
| **스마트 라우팅** | 기본 | 전체 | 전체 + 커스텀 |
| **프롬프트 캐싱** | — | 포함 | 포함 |
| **폴백** | 1단계 | 무제한 | 무제한 |
| **BYOK** | — | 포함 | 포함 |
| **지원** | 커뮤니티 | 이메일 | 전담 |
| **분석** | 기본 | 전체 | 전체 + 내보내기 |
| **팀 관리** | — | 최대 5명 | 무제한 |

## 토큰 가격

모델 비용은 제공업체 요금으로 패스스루됩니다. [모델 API](/ko/api-reference/models-list) 또는 [대시보드](https://openmodex.com/dashboard)에서 실시간 가격을 확인하세요.

### 인기 모델

| 모델 | 입력 (100만 토큰당) | 출력 (100만 토큰당) |
|-------|---------------------:|----------------------:|
| `gpt-4o` | $2.50 | $10.00 |
| `gpt-4o-mini` | $0.15 | $0.60 |
| `claude-3.5-sonnet` | $3.00 | $15.00 |
| `claude-3-haiku` | $0.25 | $1.25 |
| `gemini-2.0-flash` | $0.10 | $0.40 |
| `gemini-1.5-pro` | $1.25 | $5.00 |
| `deepseek-chat` | $0.14 | $0.28 |
| `mistral-large` | $2.00 | $6.00 |
| `text-embedding-3-small` | $0.02 | — |

### 캐시 응답

프롬프트 캐싱은 비용을 크게 절감합니다:

| | 표준 | 캐시 |
|--|--------:|-------:|
| 입력 토큰 | 100% | ~50% |
| 출력 토큰 | 100% | $0 |
| 지연 시간 | 일반 | <10ms |

## 비용 추적

모든 API 응답에는 비용 데이터가 포함됩니다:

```json
{
  "usage": {
    "prompt_tokens": 150,
    "completion_tokens": 50,
    "total_tokens": 200,
    "cost_usd": 0.001
  }
}
```

## 잔액 및 결제

- 신용카드 (Stripe) 또는 암호화폐로 **자금 추가**
- [대시보드](https://openmodex.com/dashboard)에서 **잔액 확인** 및 거래 내역 조회
- 잔액 부족 알림 **설정**
- Pro 및 Enterprise 티어에서 **자동 충전** 가능

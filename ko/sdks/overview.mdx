## 공식 SDK

OpenModex는 모든 주요 플랫폼용 공식 SDK를 제공합니다:

<CardGroup cols={2}>
  <Card title="Node.js / TypeScript" icon="js" href="/ko/sdks/node">
    `npm install @openmodex/sdk`
  </Card>
  <Card title="Python" icon="python" href="/ko/sdks/python">
    `pip install openmodex`
  </Card>
  <Card title="Go" icon="golang" href="/ko/sdks/go">
    `go get github.com/openmodexlabs/openmodex-sdk-go`
  </Card>
  <Card title="React" icon="react" href="/ko/sdks/react">
    `npm install @openmodex/react`
  </Card>
</CardGroup>

## OpenAI SDK 호환성

이미 OpenAI SDK를 사용하고 있나요? 기본 URL을 변경하여 OpenModex와 함께 사용할 수 있습니다:

<CodeGroup>
```typescript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',
  baseURL: 'https://api.openmodex.com/v1',
});
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="omx_sk_...",
    base_url="https://api.openmodex.com/v1",
)
```
</CodeGroup>

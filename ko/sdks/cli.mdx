<Note>
  OpenModex CLI는 현재 초기 개발 단계에 있습니다. 핵심 기능이 곧 제공될 예정입니다.
</Note>

## 설치

```bash
npm install -g @openmodex/cli
```

## 사용법

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## 계획된 기능

| 명령어 | 설명 | 상태 |
|---------|-------------|--------|
| `openmodex chat` | 터미널에서 대화형 채팅 | 준비 중 |
| `openmodex models` | 모델 목록 및 비교 | 준비 중 |
| `openmodex config` | API 키 및 설정 관리 | 준비 중 |
| `openmodex playground` | 빠른 프롬프트 테스트 | 준비 중 |

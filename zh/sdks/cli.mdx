<Note>
  OpenModex CLI 目前处于早期开发阶段。核心功能即将推出。
</Note>

## 安装

```bash
npm install -g @openmodex/cli
```

## 使用

```bash
# Check version
openmodex --version

# Show help
openmodex --help
```

## 计划功能

| 命令 | 描述 | 状态 |
|---------|-------------|--------|
| `openmodex chat` | 终端交互式聊天 | 即将推出 |
| `openmodex models` | 列出和对比模型 | 即将推出 |
| `openmodex config` | 管理 API 密钥和设置 | 即将推出 |
| `openmodex playground` | 快速提示词测试 | 即将推出 |

## OpenAI SDK 兼容性

已经在使用 OpenAI SDK？只需更改 base URL 即可与 OpenModex 配合使用：

<CodeGroup>
```typescript Node.js
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',
  baseURL: 'https://api.openmodex.com/v1',
});
```

```python Python
from openai import OpenAI

client = OpenAI(
    api_key="omx_sk_...",
    base_url="https://api.openmodex.com/v1",
)
```
</CodeGroup>

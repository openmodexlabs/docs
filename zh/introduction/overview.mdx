## 通用 AI 网关

OpenModex 是一个统一的 API 网关，通过单一的 OpenAI 兼容 API，为您提供来自所有主要供应商的 100 多个大语言模型的访问。

无需管理多个 API 密钥、不同的 SDK 和每个 AI 供应商的独立计费，OpenModex 让您可以：

- **使用一个 API 密钥** 访问 OpenAI、Anthropic、Google Gemini、DeepSeek、Mistral、Qwen 等
- **即时切换模型** — 只需更改一个参数，无需重构整个代码库
- **智能路由** — 让 OpenModex 根据成本、延迟或质量选择最佳模型
- **优雅降级** — 自动故障转移确保您的应用在供应商宕机时依然正常运行
- **节省费用** — 提示词缓存、成本优化路由和透明定价

## 工作原理

<Steps>
  <Step title="获取您的 API 密钥">
    在 [openmodex.com](https://openmodex.com) 注册并从控制面板创建 API 密钥。
  </Step>
  <Step title="将代码指向 OpenModex">
    将供应商的 base URL 替换为 `https://api.openmodex.com/v1`。如果您已经在使用 OpenAI SDK，只需更改一行代码。
  </Step>
  <Step title="选择任意模型">
    按名称使用任何受支持的模型 — `gpt-4o`、`claude-3.5-sonnet`、`gemini-2.0-flash`、`deepseek-chat` 等。
  </Step>
</Steps>

## 兼容 OpenAI

如果您已经在使用 OpenAI API，切换到 OpenModex 非常简单：

```typescript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'omx_sk_...',                      // your OpenModex key
  baseURL: 'https://api.openmodex.com/v1',   // point to OpenModex
});

// Everything else stays the same
const response = await client.chat.completions.create({
  model: 'gpt-4o',  // or any other model
  messages: [{ role: 'user', content: 'Hello!' }],
});
```

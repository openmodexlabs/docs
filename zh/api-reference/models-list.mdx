## 示例

<CodeGroup>
```bash cURL
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

```typescript Node.js
const models = await client.models.list();
for (const model of models.data) {
  console.log(`${model.id} — ${model.name} (${model.provider})`);
}
```

```python Python
models = client.models.list()
for model in models.data:
    print(f"{model.id} — {model.name} ({model.provider})")
```

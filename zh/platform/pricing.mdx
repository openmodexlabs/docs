## 定价方式

OpenModex 采用**按量付费**定价。您只需为消耗的 token 付费，费率透明透传。

<Info>
  OpenModex 收取与底层供应商相同的每 token 费率。模型成本没有任何隐性加价。
</Info>

## Token 定价

模型成本按供应商费率透传。通过 [Models API](/zh/api-reference/models-list) 或[控制面板](https://openmodex.com/dashboard)查看实时定价。

### 热门模型

| 模型 | 输入（每 100 万 token） | 输出（每 100 万 token） |
|-------|---------------------:|----------------------:|
| `gpt-4o` | $2.50 | $10.00 |
| `gpt-4o-mini` | $0.15 | $0.60 |
| `claude-3.5-sonnet` | $3.00 | $15.00 |

| `text-embedding-3-small` | $0.02 | — |

### 缓存响应

提示词缓存可显著降低成本：

| | 标准 | 缓存 |
|--|--------:|-------:|
| 输入 token | 100% | ~50% |
| 输出 token | 100% | $0 |
| 延迟 | 正常 | &lt;10ms |

## 成本追踪

每个 API 响应都包含成本数据：

```json
{
  "usage": {
    "prompt_tokens": 150,
    "completion_tokens": 50,
    "total_tokens": 200,
    "cost_usd": 0.001
  }
}
```

## 余额与支付

- 通过信用卡（Stripe）或加密货币**充值**
- 在[控制面板](https://openmodex.com/dashboard)中**查看余额**和交易记录
- **设置提醒**接收低余额通知
- Pro 和 Enterprise 套餐支持**自动充值**

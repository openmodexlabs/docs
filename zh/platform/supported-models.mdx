## 概述

OpenModex 通过单一 API 提供来自 6 个主要 AI 供应商的 100 多个模型。所有模型均可通过相同的 OpenAI 兼容端点访问。

<Tip>
  使用 [Models API](/zh/api-reference/models-list) 获取实时定价和可用性信息，或使用 [Compare API](/zh/api-reference/models-compare) 并排对比模型。
</Tip>

## 选择模型

使用路由策略或参考以下快速指南：

| 优先级 | 推荐 |
|----------|-------------|
| **最便宜** | `gpt-4o-mini`、`deepseek-chat`、`gemini-2.0-flash` |
| **最快** | `gemini-2.0-flash`、`gpt-4o-mini`、`claude-3-haiku` |
| **最高质量** | `gpt-4o`、`claude-3.5-sonnet`、`gemini-1.5-pro` |
| **最擅长代码** | `claude-3.5-sonnet`、`deepseek-reasoner`、`codestral` |
| **最长上下文** | `gemini-1.5-pro`（2M）、`gemini-2.0-flash`（1M） |
| **最强推理** | `o3`、`o1`、`deepseek-reasoner` |

或让 OpenModex 自动决定：

```typescript
const response = await client.chat.completions.create({
  model: 'auto',
  messages: [{ role: 'user', content: 'Hello!' }],
  routing: { strategy: 'cost_optimized' },
});
```

## 实时模型数据

获取实时定价、可用性和质量评分：

```bash
# List all models with pricing
curl https://api.openmodex.com/v1/models \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"

# Compare specific models
curl "https://api.openmodex.com/v1/models/compare?models=openai/gpt-4o,anthropic/claude-3.5-sonnet" \
  -H "Authorization: Bearer omx_sk_YOUR_KEY"
```

## 2026 年 3 月

### v1.4.0 — 模型对比 API

<Info>发布于 2026 年 3 月 15 日</Info>

- **新增：** `GET /v1/models/compare` 端点，用于并排评估模型
- **新增：** 模型响应中的质量评分（综合、代码、数学、推理）
- **新增：** 每个模型的性能指标（延迟 p50/p99、可用性）
- **优化：** 模型列表现在包含定价明细（官方费率 vs OpenModex 费率）

### v1.3.0 — BYOK 支持

<Info>发布于 2026 年 3 月 1 日</Info>

- **新增：** 自带密钥 — 使用您自己的供应商 API 密钥
- **新增：** 使用 AWS KMS 信封加密存储供应商密钥
- **新增：** 控制面板中的供应商密钥管理
- **优化：** 路由引擎处理 BYOK + 托管混合配置

## 2026 年 2 月

### v1.2.0 — React Hooks

<Info>发布于 2026 年 2 月 15 日</Info>

- **新增：** `@openmodex/react` 包，包含 `useChat`、`useCompletion`、`useModels` hooks
- **新增：** React hooks 内置流式传输和中止支持
- **优化：** TypeScript SDK 现在导出所有类型

### v1.1.0 — 提示词缓存

<Info>发布于 2026 年 2 月 1 日</Info>

- **新增：** 服务端提示词缓存，支持可配置 TTL（60 秒至 24 小时）
- **新增：** 响应元数据中的 `cache_hit` 字段
- **优化：** 缓存响应按 50% 输入费率计费，零输出成本

---

## 2026 年 1 月

### v1.0.0 — 正式发布

<Info>发布于 2026 年 1 月 15 日</Info>

- **发布：** OpenModex API 网关 — 面向 100 多个模型的 OpenAI 兼容 API
- **发布：** 智能路由（成本、延迟、质量策略）
- **发布：** 自动故障转移与熔断器
- **发布：** Node.js、Python 和 Go 的官方 SDK
- **发布：** CLI 工具（`@openmodex/cli`）
- **发布：** 包含用量分析和计费的控制面板
- 供应商：OpenAI、Anthropic、Google Gemini、DeepSeek、Mistral、Qwen

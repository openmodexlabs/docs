---
title: "Create Chat Completion"
description: "POST /v1/chat/completions — Create a model response for a conversation."
api: "POST https://api.openmodex.com/v1/chat/completions"
---

## Request Body

<ParamField body="model" type="string" required>
  Model ID to use (e.g., `gpt-4o`, `claude-3.5-sonnet`, `gemini-2.0-flash`).
</ParamField>

<ParamField body="messages" type="array" required>
  A list of messages comprising the conversation.
  <Expandable title="Message object">
    <ParamField body="role" type="string" required>
      One of `system`, `user`, `assistant`, or `tool`.
    </ParamField>
    <ParamField body="content" type="string" required>
      The content of the message.
    </ParamField>
    <ParamField body="name" type="string">
      An optional name for the participant.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="stream" type="boolean" default={false}>
  If `true`, returns a stream of Server-Sent Events (SSE).
</ParamField>

<ParamField body="temperature" type="number" default={1}>
  Sampling temperature between 0 and 2. Lower values are more deterministic.
</ParamField>

<ParamField body="max_tokens" type="integer">
  Maximum number of tokens to generate.
</ParamField>

<ParamField body="top_p" type="number" default={1}>
  Nucleus sampling parameter.
</ParamField>

<ParamField body="stop" type="string | string[]">
  Stop sequence(s) that will halt generation.
</ParamField>

<ParamField body="frequency_penalty" type="number" default={0}>
  Penalty for token frequency (-2.0 to 2.0).
</ParamField>

<ParamField body="presence_penalty" type="number" default={0}>
  Penalty for token presence (-2.0 to 2.0).
</ParamField>

<ParamField body="tools" type="array">
  A list of tools the model may call (function calling).
</ParamField>

<ParamField body="tool_choice" type="string | object">
  Controls which tool is called. `auto`, `none`, or a specific tool.
</ParamField>

<ParamField body="response_format" type="object">
  Force a specific output format (e.g., `{"type": "json_object"}`).
</ParamField>

<ParamField body="seed" type="integer">
  Deterministic sampling seed for reproducibility.
</ParamField>

### OpenModex Extensions

<ParamField body="routing" type="object">
  Configure intelligent routing for this request.
  <Expandable title="Routing object">
    <ParamField body="strategy" type="string">
      `cost_optimized`, `latency_optimized`, or `quality_optimized`.
    </ParamField>
    <ParamField body="fallback" type="string[]">
      Server-side fallback model chain.
    </ParamField>
    <ParamField body="allow_upgrade" type="boolean">
      Allow routing to a better model at the same price tier.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="cache" type="object">
  Configure prompt caching.
  <Expandable title="Cache object">
    <ParamField body="enabled" type="boolean">
      Enable prompt caching for this request.
    </ParamField>
    <ParamField body="ttl" type="integer">
      Cache time-to-live in seconds.
    </ParamField>
  </Expandable>
</ParamField>

## Example

<CodeGroup>
```bash cURL
curl https://api.openmodex.com/v1/chat/completions \
  -H "Authorization: Bearer omx_sk_YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "What is the capital of France?"}
    ],
    "temperature": 0.7,
    "max_tokens": 100
  }'
```

```typescript Node.js
import OpenModex from '@openmodex/sdk';

const client = new OpenModex({ apiKey: 'omx_sk_YOUR_KEY' });

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [
    { role: 'system', content: 'You are a helpful assistant.' },
    { role: 'user', content: 'What is the capital of France?' },
  ],
  temperature: 0.7,
  max_tokens: 100,
});

console.log(response.choices[0].message.content);
```

```python Python
from openmodex import OpenModex

client = OpenModex(api_key="omx_sk_YOUR_KEY")

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What is the capital of France?"},
    ],
    temperature=0.7,
    max_tokens=100,
)

print(response.choices[0].message.content)
```
